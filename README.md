🚗 Stand de Automóveis (JDM)
Projeto desenvolvido em Python que simula um stand de automóveis focado em carros JDM (Japanese Domestic Market). O sistema permite registo de utilizadores, visualização de stock, compra de veículos e gestão de perfil.

📌 Funcionalidades
🔐 Sistema de login e registo de utilizadores
💰 Gestão de saldo (carregar dinheiro)
🚗 Visualização de carros disponíveis
🛒 Compra de carros
👤 Gestão de perfil (editar / apagar conta)
🎨 Arte ASCII para cada modelo de carro
📦 Sistema modular (separação em ficheiros)
🧠 Estrutura do Projeto
O projeto está dividido em vários módulos:

main.py → Interface principal e menus
Carros.py → Gestão dos carros (CRUD + associação de dono)
Utilizador.py → Gestão de clientes (CRUD + saldo + carros)

⚙️ Tecnologias Utilizadas
Python 3
colorama → cores no terminal
winsound → efeitos sonoros (Windows)
Programação modular
📝 Notas
O programa corre no terminal (CLI)
O módulo winsound funciona apenas em Windows
Os dados não são persistentes (guardados apenas durante execução)
